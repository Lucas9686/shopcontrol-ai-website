## v1 Requirements

### Navigation & Layout

- [x] **NAV-01**: User can navigate between pages via responsive navbar
- [x] **NAV-02**: User can switch language between DE and EN
- [x] **NAV-03**: User sees animated hero section with project headline and CTA
- [x] **NAV-04**: User sees footer with project info, HTL-Logo, links
- [x] **NAV-05**: User experiences smooth scroll and Framer Motion animations

### Projekt-Inhalt

- [x] **CONT-01**: User can read problem statement (why small shops need AI support)
- [x] **CONT-02**: User sees feature overview (Support Agent + Buchhaltung) with cards/icons
- [x] **CONT-03**: User can explore technical deep-dive (6 workflow steps, RAG, tools, security) with diagrams/screenshots
- [x] **CONT-04**: User can view results, market differentiation, and financial projections with charts

### Extras & Meta

- [x] **META-01**: User can visit about page (Lucas Nessel, HTL Klagenfurt, Betreuer)
- [x] **META-02**: User can access Impressum/Datenschutz page
- [x] **META-03**: Page has proper SEO meta tags and Open Graph image for social sharing
- [x] **META-04**: User can watch embedded demo video showing the agent in action

### Technische Basis

- [x] **TECH-01**: Site uses Next.js + shadcn/ui + Tailwind v4 + Framer Motion
- [x] **TECH-02**: i18n routing with [locale] pattern (DE + EN)
- [x] **TECH-03**: Optimized images (Next.js Image component for all screenshots)
- [x] **TECH-04**: Dark blue SaaS-style design theme
- [x] **TECH-05**: Deployable on Vercel
